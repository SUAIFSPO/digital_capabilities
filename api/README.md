# Database
Это приложение является api для работы с базой данных посредством http запросов. Она уже располагается по адресу http://3.17.59.226, но визуальной составляющей не имеет, а необходима для работы остальных приложений. Если необходимо развернуть базу локально, то нужно перейти в папку api/DatabaseAPI/bin/Debug/netcoreapp3.1/ и открыть файл DatabaseAPI.exe. База настроена на прием запросов только с локального компьютера, поэтому рекомендуется использовать базу на сервере.
# Тестовые пользователи
Для  проверки работоспособности системы созданы тестовые пользователи (логин:пароль:кодовое слово):
```
admin:admin:recword1 - администратор
user:user1:recword2 - пользователь
```
Также были добавлны 30 тестовых пользователей вида user10, user11.. user39. Все они принадлежат 3ем группам (первая цифра в логине). Для проверки нейроной сети необходимо использовать пользователей одной группы (user10, user11, ..., user19). Пароль оот всех этих записей: user
