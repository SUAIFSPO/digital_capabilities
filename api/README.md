# Database
Это приложение является api для работы с базой данных посредством http запросов. Она уже располагается по адресу http://3.17.59.226, но визуальной составляющей не имеет а необходима для работы остальных приложений. Если необходимо развернуть базц локально, то необходимо перейти в папку api/DatabaseAPI/bin/Debug/netcoreapp3.1/ и открыть файл DatabaseAPI.exe. База настроена на прием запросов только с локального компьютера, поэтому рекомендуется использовать базу на сервере.
